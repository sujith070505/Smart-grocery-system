# Smart Grocery Price Intelligence System

## Project Overview
A web-based grocery price comparison system developed using Java JSP, Servlets, and MySQL. The application helps users compare grocery prices across multiple platforms and identify the best deals.

## Features
- User Registration and Login
- Product Search
- Grocery Price Comparison
- Shopping Cart
- Order Management
- Admin Dashboard
- MySQL Database Integration

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
